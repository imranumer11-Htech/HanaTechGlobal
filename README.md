# HanaTech Global Website

Multi-page IT training website with:
- Detailed, engaging homepage
- Koenig-style course listing page with sidebar categories
- Course detail modal (description + brochure + direct course-page link)
- Individual landing page for every course in `courses/`
- Custom course request form

## Main Pages
- `index.html`
- `programs.html`
- `schedule.html`
- `contact.html`

## Course Landing Pages
- `courses/*.html` (auto-generated, one page per course)

## Brochure Files
- `brochures/aws-saa-exam-guide.pdf`
- `brochures/microsoft-certification-poster.pdf`
- `brochures/oracle-oci-foundations-study-guide.pdf`
- `brochures/pmi-pmp-handbook.pdf`
- `brochures/pmi-capm-handbook.pdf`
